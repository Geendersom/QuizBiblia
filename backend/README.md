# Backend - Quiz Bíblia

## Estrutura Base

Este diretório contém a estrutura base para o backend futuro do Quiz Bíblia.

## Status

Atualmente é apenas uma estrutura mock. A implementação real será feita quando necessário.

## Endpoints Planejados

- `GET /ranking` - Obter ranking online
- `POST /score` - Enviar score de um jogador
- `GET /players` - Listar jogadores

## Como Usar (Futuro)

```bash
npm install
npm run dev
```

## Notas
